# Docker

## Comandos básicos

### Crear contenedor

`docker-machine create --driver virtualbox 'nombreContenedor'`

### Configurar ambiente

`docker-machine env nombreContenedor`

### Listar contenedores

`docker-machine ps`

### Obtener ip

`docker-machine ip <nombreContenedor>`

### Listar contenedores locales

`docker-machine ls`

### Listar imágenes locales

`docker images`
